## Dependencies

### Three.js

To install three.js, download the `build` folder from the latest github release

```bash

$ cd 3p/three
. 
$ curl -L https://github.com/mrdoob/three.js/archive/refs/tags/r183.tar.gz | tar -xz --strip-components=2 three.js-r183/build
.
```
